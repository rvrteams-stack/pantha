# Keto Coffee PT Mindmap

Interactive team mindmap for Keto Coffee PT.

## How it works

- `mindmap.md` is the source outline.
- `index.html` fetches `mindmap.md` and renders it as a self-contained interactive SVG mindmap.
- Push changes to the publishing branch and GitHub Pages will update the site.

## GitHub Pages setup

In the repository settings, use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

The project URL should be:

```text
https://rvrteams-stack.github.io/pantha/
```

GitHub says Pages can publish from a branch/folder, and the source folder can be the repository root or `/docs`.

## Privacy note

Private repository visibility protects the source files from people who are not invited to the repository. A GitHub Pages site can still be public on the internet unless the organization has GitHub Enterprise Cloud private Pages access control enabled.
